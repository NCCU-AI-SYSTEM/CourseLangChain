# 2026/03/31 - 2026/04/05 Weekly Note

## 本週主題

將目前的單純 RAG 架構，逐步重構成 `Brain Agent + Tools` 的架構。

核心方向：

- 保留原本的課程檢索能力
- 將原本的 RAG 課程查詢包裝成 Brain Agent 可呼叫的一個 tool
- 後續新增功能時，以新增 tool 為主，不再把所有邏輯塞進單一 prompt 或單一 chain
